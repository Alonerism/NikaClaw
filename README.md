# NikaMolt

Minimal "command center" UI for Moltbot/OpenClaw-style agents.

## Run

```bash
npm install
npm run start
```

Then open:
- http://127.0.0.1:3999

## Goals
- Task-first governance UI (tree of tasks/subtasks)
- Transparent activity log (heartbeats/wakes + what changed)
- Designed to support connecting to multiple agent instances
