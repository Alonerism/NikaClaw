# Roadmap (high ROI)

## v0.2 (task-first)
- Task tree model: parentId, children, ordering
- Task details view (drawer): notes, artifacts, how-to-test
- "Human needed" flag only when required (keys/DNS/OAuth)
- Events auto-generated from task transitions + agent activity

## v0.3 (instances)
- Instances page: connect multiple agent instances
- Per-instance health (last heartbeat, last action, status)
- Token storage + proxy settings

## v0.4 (intern handoff)
- Export/share: generate a handoff doc for a task (markdown/pdf)
- Assign tasks to a human (owner field)

## v0.5 (security)
- Auth on dashboard
- Read-only viewer role
